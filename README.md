# Hubbard Heights Men's Golf Club

Website for the Hubbard Heights Men's Golf Club, Stamford, Connecticut.

## Stack

- Static HTML/CSS/JS — single `index.html` file
- Hosted on GitHub Pages (free, auto-deploys on push)
- Domain managed via Wix (DNS pointing to GitHub Pages)
- PayPal buttons for membership payments

## How to Update

1. Edit `index.html` directly in GitHub (click the file, then the pencil icon)
2. Commit changes — site auto-deploys in ~60 seconds
3. Or use the CMS at `/admin` for content updates (coming soon)

## Structure

- Hero — 2025 Interclub Champions photo + call to action
- About — Club description + Boomerang film trivia
- Sponsors — MBA Graphics + Bob Judge Insurance
- Photos — Season gallery
- Membership — 4 tiers (Regular $95, Senior $75, Associate $65, Junior $40)
- Officers — Club officers list
- News — Latest club news
- Contact — Address, phone, email, contact form
